Horos Plugin

ADC Map

Version 2.0

About this Plugin:

This is a simple plugin that allows you to calculate an ADC map (apparent diffusion coefficient) from images acquired with different diffusion weighting (b values). Simply open multiple images in the 4D viewer and run the plugin.

Notes:

  1. The plugin attempts to read b values from image headers. However, there is little consistency as to how this information is stored, so we will continue to work on this, and appreciate your feedback.
  2. You can instead just enter the b values for the acquired images in the plugin, which works fine.
  3. Maps are grayscale by default, but you can change the CLUT to make them color (etc).
